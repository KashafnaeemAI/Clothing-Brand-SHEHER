# Clothing-Brand-SHEHER
Modern clothing brand website created using AI tools as a 2nd semester BS AI student project. It includes a home page, product gallery, about us, contact page, and shopping cart UI. The design is minimalist, responsive, and user-friendly, focusing on elegant fashion presentation and basic web development skills.
AI Tool Used:Loveable AI
Prompt Used:"Create a professional and aesthetic clothing brand website for'Luxe Lawn Elegance' featuring a product gallery,navigation bar,and a premium summer collection theme".
Key Features:
Modern Home Page &Product Gallery.
Shopping Cart UI &Contact Page.
Responsive design for all devices.
